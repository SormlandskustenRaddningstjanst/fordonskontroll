# Fordonskontroll

App för fordons- och materialkontroller inom räddningstjänst.

## Funktioner
- Daglig kontroll
- Veckokontroll
- Månadskontroll
- Kvartalskontroll
- Avvikelser
- Instruktioner och video per kontrollpunkt

## Struktur
- backend/ API
- database/ SQL schema
